# Supply App
Программа нужна для учета поступающих заявок на закупку определенных товаров.

## Требования:

- .NET 6 [.NET Desktop Runtime 6.0.14](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)
- 1 ГБ свободного места
- 2 ГБ ОЗУ

## Стандартные данные
